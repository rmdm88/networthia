# Capital Atlas

MVP of a personal and family net worth tracker built with Next.js App Router, Tailwind CSS and Recharts.

## Local run

```bash
npm install
npm run dev
```

If dev hot-reload gets stuck, run:

```bash
npm run dev:reset
```

## Included

- Snapshot-based net worth tracking without transaction imports
- Personal and family dashboard in RUB
- Multi-folder account grouping
- Digest feed with de-duplicated posts
- Dark theme toggle
- Client-side localStorage persistence

## Deploy

```bash
npm run build
```
